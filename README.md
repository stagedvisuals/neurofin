## 🚀 Snel Starten

### 1. Frontend Development
```bash
cd frontend
npm install
npm run dev
```

### 2. Build voor Productie
```bash
cd frontend
npm run build
npm start
```

### 3. Environment Variabelen
```bash
# .env.local
NEXT_PUBLIC_API_URL=http://localhost:8000
NEXT_PUBLIC_QUANTUM_SIMULATOR=true
NEXT_PUBLIC_AR_ENABLED=true
```
